신입 웹퍼블리셔,
조혜진 입니다 👋

저는 코딩으로 웹을 표현하는 것에 매우 흥미를 느껴 웹퍼블리셔를 꿈꾸기 시작하였습니다. 웹표준 및 웹접근성과 시멘틱한 마크업을 중시하고, 보다 견고하면서 사용자에게 편리한 UI/UX 를 추구하는 웹퍼블리셔 입니다.
또한 자바스크립트를 사용한 동적인 웹 구현에 관심이 많으며, 정체 되어 있지 않고 항상 끊임없는 발전을 위해 꾸준히 공부를 하며 노력중입니다!

Skill

  • HTML5
  • CSS
  • SCSS
  • Javascript
  • jQuery

  • 시멘틱한 구조와 웹표준 및 웹 접근성에 준수하는 마크업을 합니다. HTML 은 물론 모바일 페이지를 하드코딩할 수 있습니다.

  • 주어진 디자인을 자유자재로 구현하며 CSS 속성에 대한 이해가 깊습니다.
    미디어 쿼리를 이용한 반응형 레이아웃을 구사할 수 있습니다.

  • SCSS 문법을 통해 기존의 CSS 보다 효율적으로 스타일링하며, 불필요한 부모 선택자의 남용을 줄일 수 있습니다.

  • 자바스크립트에 대한 이해도가 있으며 간단한 기능들을 하드코딩할 수 있습니다.
    이를 통해 라이브러리나 플러그인을 보다 수월하게 사용할 수 있습니다.

  • 이벤트 메서드를 사용해 기능을 구현하는 동시에 다양한 플러그인을 활용할 수 있습니다.
